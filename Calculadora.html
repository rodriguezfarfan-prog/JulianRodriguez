🎮 Explicación del Juego: Calculadora

Nombre: Calculadora Interactiva

Lógica de Funcionamiento:

  • Utiliza tres variables principales: current (número actual), previous (número anterior) y operator (operador seleccionado)
  • Al presionar números, se concatenan al valor actual
  • Al seleccionar un operador, se guarda el número actual como "anterior" y se prepara para el siguiente número
  • Al presionar "=", se ejecuta la operación matemática correspondiente usando switch/case
  • Incluye validaciones para evitar múltiples puntos decimales y división por cero

Retos Técnicos:

  • Manejo correcto del estado entre operaciones encadenadas
  • Implementación de soporte para teclado físico
  • Validación de entrada de decimales
  • Actualización en tiempo real del display

Aprendizajes Obtenidos:

  • Manipulación del DOM con JavaScript
  • Manejo de eventos (click y keyboard)
  • Control de estados y variables globales
  • Validación de datos de entrada
  • Diseño de interfaz de usuario intuitiva

💡 Código Comentado - Función Principal

// Función para realizar el cálculo
function calculate() {
    // Validar que tenemos todos los datos necesarios
    if (operator === null || previous === '' || current === '') return;
    
    // Convertir strings a números
    const prev = parseFloat(previous);
    const curr = parseFloat(current);
    let result;
    
    // Realizar la operación según el operador
    switch(operator) {
        case '+':
            result = prev + curr;  // Suma
            break;
        case '-':
            result = prev - curr;  // Resta
            break;
        case '*':
            result = prev * curr;  // Multiplicación
            break;
        case '/':
            // Validar división por cero
            result = curr !== 0 ? prev / curr : 'Error';
            break;
    }
    
    // Actualizar el display con el resultado
    current = result.toString();
    operator = null;
    previous = '';
    updateDisplay();
}
